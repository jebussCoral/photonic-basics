# Photonic Basics

This repository documents a step-by-step learning process in integrated photonics.

The goal is to build intuition and practical skills starting from basic optical models (MZI) up to more advanced architectures (logic, AI accelerators).

---

## Day 1 — Ideal Mach-Zehnder Interferometer (MZI)

We implemented a simple MZI model based on interference of optical fields:

E_out = E1 + E2 * exp(i * phi)

I_out = |E_out|^2

### Key concepts

- Optical field vs optical power  
- Phase difference (phi)  
- Interference (constructive and destructive)  
- Perfect extinction (ideal case)  

### Results

- Maximum intensity at phi = 0  
- Minimum intensity at phi = pi  
- Periodicity of 2π
